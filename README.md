## Descripción

**EduPork** es una aplicación web desarrollada en **Python con Flask y JavaScript**, diseñada para optimizar la **gestión, formulación y análisis de dietas porcinas**. Integra procesos técnicos del área agropecuaria con soluciones tecnológicas, permitiendo formular dietas balanceadas, analizar nutrientes y registrar información de manera sencilla y organizada.

---

## Objetivo del proyecto

Desarrollar una solución tecnológica que apoye procesos académicos y productivos del sector agropecuario, facilitando el manejo de información nutricional porcina mediante una plataforma web intuitiva y eficiente.

---

## Funcionalidades

* Formulación de dietas porcinas balanceadas
* Análisis nutricional de alimentos
* Registro y gestión de dietas
* Cálculos automáticos de nutrientes
* Interfaz web amigable
* Persistencia de datos en base de datos

## Instalación y ejecución

1. Clonar el repositorio

```bash
git clone https://github.com/usuario/edupork.git
```

2. Crear y activar entorno virtual

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

3. Instalar dependencias

```bash
pip install -r requirements.txt
```

4. Configurar la base de datos

* Importar el archivo `Script bd edupork.sql` en MySQL

5. Ejecutar la aplicación

```bash
python app.py
```
